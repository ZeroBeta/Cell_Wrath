# Cell

Clean, responsive raid and party frames designed to surface the information you need without the clutter.

**Highlights**
- Inspired by CompactRaid, Grid2, Aptechka, and VuhDo, refined into a friendlier, faster package.
- Built for clarity: sharp debuff tracking, healer-friendly indicators, and quick setup.
- Works great on Ascension and Wrath-era clients.

**Ascension Wrath port**
- Ported by ZB.
- Improvements & bugs: Discord `ZB#08374` or in-game on Bronzebeard (`Zeruto` / `Zeruta` / `Reana`).
- Port date: 2025-12-03.

## Features

- __Layouts:__ auto switch layout by spec/role, supports party, raid, arena, and battleground.
- __Customizable Appearance:__ textures, colors and alphas.
- __Built-in Click-Castings:__ supports keyboard and multi-button mouse.
- __Indicators:__Â dozens of built-in indicators and unlimited custom indicators (icon, bar, rect, text, icons).
- __Raid Debuffs:__ debuffs priority and glow.
- __Useful Raid Tools:__ ready check, countdown, rebuff, death report, marks, battleres.
- __Nice Options UI:__ I mean yes it's pretty darn good!
- __Spotlight Frame:__ extra 15 unit buttons, can be set to Target, Focus, Unit, Tank, etc.
